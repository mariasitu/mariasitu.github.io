Dropbox Redesign Challenge

Given 24 hours, I was tasked to redesign the Dropbox homepage to increase the visibility of the types of files that users have.
Overview

Challenge

As a part of Bloc's Apprenticeship program, I was given a 24 hour challenge to redesign Dropbox's dashboard. The goal of the challenge was to help increase the visibility of the types of files users have.


Solution

After conducting some quick user interviews and user testing, I created a dashboard that shows users the total amount of files in their account with specific separations for each file type: Documents, Spreadsheets, Images, and Media files.

  • Deliverables

  • User surveys, User personas, User stories, Wireframes, High-fidelity mockup
